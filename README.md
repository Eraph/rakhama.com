# Rakhama.com

## [Hugo](https://gohugo.io/)
### Create new site
``` bash
hugo new site <sitename>
```
### Create new post
``` bash
hugo new posts/<post-name>.md
```

### Create new page
``` bash
hugo new pages/<page-name>.md
```

### Compile
``` bash
hugo
```

### Compile with drafts
``` bash
hugo -D
```
